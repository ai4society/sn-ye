SN-YE: A tool for Surya Namaskara - Yoga Exploration

This is a tool to explore Surya Namaskar (SN), or Sun Salutation, which is a set of choreographed asanas from Yoga. The tool is powered by making queries on the SN-YO ontology.

Image Credit: all visual references in the tool are taken from Common Yoga Protocol (English), Ministry of Ayush, Government of India .

Natural Language Interface

Query the Knowledge Graph

This framework interprets your natural language prompts and translates them into structured SPARQL queries against the Surya Namaskar Ontology.

Gemini API Key & Model Required
A Gemini API key is required to generate SPARQL and the natural language explanation. Open the Gemini API Key & Model section to add it.

Getting Started

Example Queries

Select a model and enter a natural language prompt on the left to begin generating custom queries dynamically. If you're not sure where to start, try these tested examples:

  • "What are the different variants of SN?"
  • "What is the ordered sequence of poses in the Base Surya Namaskar (Sivananda) variant?"
  • "What rules and common errors are documented for Bhujangasana (Pose 7)?"
  • "What breathing pattern and safety note are modeled for each Base SN pose?"
  • "Which poses in Base SN carry mantra and chakra annotations?"
  • "Compare the pose counts across all three variants (Base SN, V1, V2, V3)."
  • "Which asanas are shared across multiple variants?"
  • "What is the ordered pose sequence for V1 (Krishnamacharya)?"
  • "Which poses exhibit inverse left/right laterality pairings?"
  • "Which asanas have linked CYP (Common Yoga Protocol) visual references?"
