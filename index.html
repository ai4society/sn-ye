SN-YE: A tool for Surya Namaskara - Yoga Exploration

This is a tool to explore Surya Namaskar (SN), or Sun Salutation, which is a set of choreographed asanas from Yoga. The tool is powered by making queries on the SN-YO ontology.

Image Credit: all visual references in the tool are taken from Common Yoga Protocol (English), Ministry of Ayush, Government of India .

Try queries Predefined, SPARQL, or Natural Language

Start from validated competency questions, write SPARQL directly, or ask supported questions through Gemini-assisted templates.

Inspect evidence Ontology facts and source queries

Review retrieved rows, generated SPARQL, linked asanas, breathing annotations, safety notes, and visual references.

Language Localized asana labels

Use the Language selector to add available non-English asana labels to predefined and natural-language results.

Natural Language Interface

Query the Knowledge Graph

Getting Started

Example Queries

Select a model and enter a natural language prompt on the left to begin generating custom queries dynamically. If you're not sure where to start, try these tested examples:

  • "What are the different variants of SN?"
  • "What is the ordered sequence of poses in the Base Surya Namaskar (Sivananda) variant?"
  • "What rules and common errors are documented for Bhujangasana (Pose 7)?"
  • "What breathing pattern and safety note are modeled for each Base SN pose?"
  • "Which poses in Base SN carry mantra and chakra annotations?"
  • "Compare the pose counts across all three variants (Base SN, V1, V2, V3)."
  • "Which asanas are shared across multiple variants?"
  • "What is the ordered pose sequence for V1 (Krishnamacharya)?"
  • "Which poses exhibit inverse left/right laterality pairings?"
  • "Which asanas have linked CYP (Common Yoga Protocol) visual references?"

Query Examples

                    
SPARQL Templates for SN-YO

                  

                


                

                  Use these examples as starting points for direct ontology exploration. Paste one into the editor above after the prefix block, or adapt the variables and predicates for your own question.
                


                

                  

                    Base SN sequence with asanas and breathing
                    
SELECT ?pose ?poseNumber ?asanaLabel ?breathingLabel
WHERE {
  ?pose rdf:type core:Pose .
  ?pose core:belongsToVariant base:BaseSN_SivanandaYogaVedantaCentre_UsedatIITBHU .
  ?pose core:poseNumber ?poseNumber .
  ?pose core:hasAsana ?asana .
  ?asana rdfs:label ?asanaLabel .
  FILTER(LANG(?asanaLabel) = "en")
  OPTIONAL {
    ?pose core:hasBreathingPattern ?breathing .
    ?breathing rdfs:label ?breathingLabel .
    FILTER(LANG(?breathingLabel) = "en")
  }
}
ORDER BY ?poseNumber

                  


                  

                    Available asanas with localized labels
                    
SELECT ?asana ?englishLabel ?localizedLabel
WHERE {
  VALUES ?language { "hi" }
  ?asana rdf:type core:Asana .
  ?asana rdfs:label ?englishLabel .
  FILTER(LANG(?englishLabel) = "en")
  OPTIONAL {
    ?asana rdfs:label ?localizedLabel .
    FILTER(LANG(?localizedLabel) = ?language)
  }
}
ORDER BY ?englishLabel

                  


                  

                    Safety notes in the base sequence
                    
SELECT ?poseNumber ?asanaLabel ?safetyNote
WHERE {
  ?pose rdf:type core:Pose .
  ?pose core:belongsToVariant base:BaseSN_SivanandaYogaVedantaCentre_UsedatIITBHU .
  ?pose core:poseNumber ?poseNumber .
  ?pose core:hasAsana ?asana .
  ?asana rdfs:label ?asanaLabel .
  FILTER(LANG(?asanaLabel) = "en")
  ?pose core:hasSafetyNote ?safetyNote .
}
ORDER BY ?poseNumber

                  


                  

                    Asanas with CYP visual references
                    
SELECT ?asana ?asanaLabel ?cypPage
WHERE {
  ?asana rdf:type core:Asana .
  ?asana rdfs:label ?asanaLabel .
  FILTER(LANG(?asanaLabel) = "en")
  ?asana core:hasCYPPage ?cypPage .
}
ORDER BY ?cypPage
